File FIR at nearest police station

Dial: 100 (Emergency)

Immediate Actions

  • Call your bank immediately to block the card
  • Change all banking passwords and PINs
  • Enable Two-Factor Authentication (2FA)
  • File complaint on Cyber Crime Portal

Post-Fraud Steps

  • Monitor credit report via CIBIL/Experian
  • Check for unauthorized transactions daily
  • Update security settings in banking apps
  • File RBI Ombudsman complaint if unresolved

Prevention Tips

  • Never share OTP/CVV with anyone
  • Use virtual cards for online payments
  • Enable transaction alerts via SMS/email
  • Regularly check account statements

Resources

  • RBI Guidelines: rbi.org.in
  • CIBIL Dispute: cibil.com
  • NPCI Portal: npci.org.in
